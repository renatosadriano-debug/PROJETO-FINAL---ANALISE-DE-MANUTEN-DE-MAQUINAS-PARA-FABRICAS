# PROJETO-FINAL---ANALISE-DE-MANUTEN-O-DE-MAQUINAS-PARA-FABRICAS
é um pipeline de Machine Learning para manutenção preditiva na Indústria 4.0. A partir de dados históricos de sensores, o sistema prevê a probabilidade de falha mecânica em equipamentos (classificação binária), permitindo intervenções antes da quebra e reduzindo paradas na linha de produção.
